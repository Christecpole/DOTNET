# Exercice Docker #6 - Déploiement d'une API en .NET

## Objectifs

Appréhender le fonctionnement de Docker dans le cadre d'une API .NET

## Sujet

Via l'utilisation de Docker Desktop, d'un fichier Dockerfile ainsi qu'une application en .NET de base de type WebAPI, réaliser son déploiement:

* Créer un projet de type WebAPI en .NET
* Ecrivez le fichier Dockerfile servant à sa mise en conteneur
* Compiler l'image de votre application
* Déployer via Docker l'applicatif (attention à ne pas oublier la liaison des ports si l'on veut tester)
* Tester l'applicatif avec un client HTTP style ThunderClient ou Postman
