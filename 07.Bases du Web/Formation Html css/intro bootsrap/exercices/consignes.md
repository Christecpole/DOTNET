### Exercice 1 :
- Créer une page `index.html` (ou `home.html`) avec Bootstrap (CDN).
- Faire une grille (= mise en page en colonnes) de 3 blocs :
  - 1 colonne sur mobile
  - 2 colonnes à partir de `md`
  - 3 colonones à partir de `lg`

Contraintes :
- utiliser `container`, `row`, `col-12 col-md-6 col-lg-4`
- utiliser `g-3`
- mettre un titre + un paragraphe dans chaque bloc

Critères de réussite :
- en redimensionnant la fenêtre, la grille (= mise en page en colonnes) se réorganise correctement

### Exercice 2 :

- Modifier cette  page `list.html` (ou une section dédiée) avec un tableau (voir sur bootstrap docs des élements à ajouter à ce tableau) :
  - colonnes : Id, Nom, Prix, Stock, Actions
  - au moins 8 lignes
  - actions par ligne : Détail / Modifier / Supprimer (boutons `btn-sm`)
