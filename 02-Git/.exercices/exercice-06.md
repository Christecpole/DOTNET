# Exercice 06 - Corrections sur branches locales / distantes

## Objectifs

Appréhender le fonctionnement des branches dans l'écosystème Git et GitHub

## Sujet

* Créer un dépot Git pour sauvegarder notre projet de développement d'application frontend
* Faire un commit initial contenant un fichier descriptif de l'applicatif et de ses objectifs
* Créer une branche servant à la création de la page d'accueil de notre site
* Faire un ensemble de commits et de modifications à notre applicatif dans la branche principale
* Oops, on s'est trompé de branche. Faire en sorte que les commits précédents se trouvent au final sur la branche nouvellement créée
* Se placer sur la branche principale et rapatrier les changements de la branche de feature
* Déposer les changements sur le dépot distant
* Retourner sur la branche de feature et coder un bogue avant de le commiter
* Faire en sorte d'annuler le bogue via un commit correctif, de sorte à ce que la branche pointe sur un commit retirant le bogue précédemment introduit
