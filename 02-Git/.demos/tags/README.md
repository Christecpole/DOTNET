# Descriptif de l'application

Pour pouvoir tester les appels API vers les clients, utiliser un client HTTP et faire appel aux endpoints:
* `GET /api/v1/clients` => Va retourner le listing des clients
* `POST /api/v1/clients` => Pour ajouter un nouveau client
* `DELETE /api/v1/clients/:clientId` => Pour supprimer un client avec son ID au format `UUID`
