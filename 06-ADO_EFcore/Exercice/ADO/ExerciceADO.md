## Exercice ADO

- Réalisez une application en C# qui demande à l'utilisateur de saisir :
    - Nom
    - Prénom
    - numéro de classe
    - Date de diplôme
- L'application ajoutera les données dans une table etudiant.

- On souhaite modifier notre application pour ajouter les fonctionnalités suivantes :
    - Afficher la totalité des etudiants.
    - Afficher les etudiants d'une classes.
    - Recuperer un Etudiant par son ID
    - Recuperer les Etudiants par leurs nom
    - Mettre a jours un Etudiant
    - Supprimer un etudiant.
